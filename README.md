# LRU-LFU-OPT-FIFO_Algorithm
一个使用Python实现 LRU&amp;LFU&amp;OPT&amp;FIFO 算法，并使用VUE+ElementUI+Flask 搭建简易客户端的项目
如何运行：服务端：python xx.py 页面端：npm run serve 

注意：
- 目前只支持0~9的输入，不支持多位数
- LFU 算法目前只考虑到出现的次数
